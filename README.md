# memes.tw

![memes taiwan](update/1624410735344.jpg)

中文迷因標註數據集

目前 memes.tw 梗圖倉庫頁數為 630，如果出現大於這個數字麻煩大大跑一下 test.py 然後更新 meme_captions.tsv 檔案

```bash
python update/download.py
```

### 需要的模組

```
requests
tqdm
beautifulsoup4
```

安裝

```bash
pip install requests tqdm beautifulsoup4
```
